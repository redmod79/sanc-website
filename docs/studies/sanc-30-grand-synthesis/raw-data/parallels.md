# Cross-Testament Parallels
Key connections established in prior studies:
- EXO 25:8 -> REV 21:3 (dwelling purpose fulfilled)
- LEV 16:12-13 -> REV 8:3-5 (censer scene)
- LEV 16:17 -> REV 15:8 (no entry)
- LEV 16:20-22 -> REV 20:1-3 (scapegoat/Satan)
- EXO 28:38 -> HEB 9:28 (priestly sin-bearing via nasa/anaphero)
- DAN 8:14 -> REV 15:3; 16:7 (vindication)
- ROM 3:25 -> HEB 9:5 (hilasterion = mercy seat)
